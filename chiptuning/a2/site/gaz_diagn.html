ПРОГРАММА ДИАГНОСТИКИ GAZ-DIAGN

Программа GAZ-DIAGN предназначена для диагностики автомобилей УАЗ и ГАЗ с двигателями, оснащёнными электронной системой управления впрыском топлива с контроллерами МИКАС-5.4, МИКАС-7 и их модификаций.

Данная программа функционирует на персональной ЭВМ типа IBM-PC/AT совместно с адаптером RS-232/K-Line, например, адаптер АПМ-1.1. Для коммутации с диагностическим разъемом автомобиля используется диагностический кабель ГАЗ, для УАЗ «Патриот» используется кабель диагностический OBD2.

Программа функционирует под управлением операционной системы MS Windows 98, MS Windows 2000 или MS Windows XP.

Все права на данную программу принадлежат ООО «А2», г. Ульяновск.

Программа распространяется бесплатно.
