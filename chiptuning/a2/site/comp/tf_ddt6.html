Датчик давления ДДТ-6М

Для просмотра увеличенного изображения, «кликните» по фотографии прибора.

Данный датчик применяется совместно с прибором ТФМ-3 для измерения давления в топливной магистрали.

Основные технические характеристики:

  1. Диапазон измеряемых давлений— 0..587,9кПа.
  2. Предельное допустимое давление— не более 979,9кПа.
  3. Погрешность в рабочем диапазоне температур— не более 1%.
  4. Диапазон рабочих температур— -30..+50°C.
