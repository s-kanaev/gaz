Программа АвтоПланирование.

Система планирования работ предназначена для планирования рабочего времени и составления расписаний на станциях технического обслуживания автомобилей, автомойках, шиномонтажных мастерских и т.п. Использование программы значительно упрощает централизованное управление планированием и распределением рабочего времени предприятия.
Планирование на Вашем предприятии поможет Вам снизить потери рабочего времени отдельных подразделений, а следовательно повысить эффективность всего предприятия в целом.

О программе АвтоПланирование.

Версия 3.0 (Что нового?)

Планирование
  • Создание неограниченного количества подразделений для планирования их работы.
  • Цветовое выделение заданий подразделениям.
  • Создание неограниченного количества собственных цветовых шаблонов для формирования заданий.
  • Создание неограниченного количества предприятий.
  • Планирование работы подразделений СТО по времени в течение дня.
  • Планирование работы подразделений СТО по дням в течение недели.
  • Планирование работы подразделений СТО по дням в течение месяца.
  • Возможность просмотра информации, как в режиме "планировщика", так и в табличном режиме.
  • Возможность фильтрации заданий по указанному критерию (номеру, времени создания, предприятию, клиенту, автомобилю).
  • Возможность отображения определенной группы заданий (выполненные/невыполненные).
  • Отображение в сплывающей подсказке содержимого задания.
  • Настройка "планировщика" (установка необходимого пользователю размера сетки, рабочего дня, периодичности и т.д.).
  • Для облегчения просмотра больших объемов информации, введена возможность скрывать отдельные объекты (в частности пользователей).
  • Быстрый ввод информации по заданиям (все операции производятся практически одним движением мыши).
  • Смена заданий между подразделениями.
  • Быстрое изменение продолжительности задания, времени начала или окончания.
  • Быстрый переход по дням, неделям и месяцам.
  • Поиск заданий в базе данных.
  • Быстрая очистка базы данных от выполненных (ненужных) заданий.
  • Вывод информации о текущем состоянии базы данных (общем количестве заданий и их принадлежности).
  • Возможность копирования и вставки уже введенных заданий.
  • Возможность установки времени и звукового оповещения для задачи.
  • Возможность просмотра связанных задач по дням, неделям, месяцам.
Ведение различных справочников
  • Справочник по клиентам с возможность установки неограниченного количества автомобилей для каждого клиента с получением статистики. Реализован механизм поиска и фильтрации клиентов.
  • Справочник моделей с получением статистики по выбранному автомобилю.
  • Справочник работ с их разделением на группы по функциональному назначению, механизмом фильтрации по наименованию и коду работы, формированием прайс-листов всего перечня услуг и статистических данных по выбранной работе.
  • Справочник материалов с их разделением на группы по функциональному назначению, механизмом фильтрации по наименованию, формированием статистических данных по выбранному материалу.
  • Справочник предприятий с возможностью формирования статистических данных по конкретному предприятию.
  • Справочник бригад и сотрудников с возможностью создания новых сотрудников, формирования из них бригад, назначения каждому члену бригады параметров его трудоучастия, тарифа, процентной ставки.
Отчеты
  • Формирование различных отчетов в разрезе указанных для них параметров, сохранение установленных параметров с их последующим восстановлением.
  • Возможность сохранять сформированные отчеты и открывать существующие.
  • Возможность просмотра и печать статистики по основным объектам системы.
  • Формирование и печать заказ-наряда для каждого задания.
  • Возможность правки отчетов через мастер отчетов.
  • Возможность установки необходимых параметров отображения отчета.
  • В системе формируются следующие отчеты:
    • "Реестр заданий". Отчет формируется в разрезе следующих параметров: временного периода, предприятия, типа задания (выполненное/невыполненное), вида оплаты (наличный/безналичный), приемщика задания, рабочего места.
    • "Отчет по работам". Отчет формируется в разрезе следующих параметров: временного периода, предприятия, типа задания (выполненное/невыполненное), с указанием исполнителей или без них.
    • "Отчет по материалам". Отчет формируется в разрезе следующих параметров: временного периода, предприятия, типа задания (выполненное/невыполненное), с возможностью рассчитывать суммы по каждому заданию.
    • "Отчет по бригадам". Отчет формируется в разрезе следующих параметров: временного периода, предприятия, типа задания (выполненное/невыполненное), с указанием бригад, по которым производить расчет.
    • "Отчет по исполнителям". Отчет формируется в разрезе следующих параметров: временного периода, предприятия, типа задания (выполненное/невыполненное), с указанием бригад, по которым производить расчет, и метода расчета заработной платы (по тарифу или по процентной ставке).
  • В системе формируется следующая статистика:
    • По клиенту;
    • По автомобилю;
    • По работе;
    • По материалу;
    • По сотруднику;
    • По предприятию;
Сервисные возможности
  • Большое количество параметров работы системы, настройка которых упростит работу.
  • Возможность работы с системой нескольких пользователей с разграничением прав доступа к различным частям программы.
  • Ведение журнала работы, в котором отражены действия, производимые пользователями с основными объектами системы.
  • Руководство пользователя, справочная система.
  • Интуитивно понятный интерфейс.
  • Возможность функционирования в сети.

Требования и рекомендации.

Программные средства
Система "АвтоПланирование" тестировалась на следующих операционных системах:
  • Windows® 98 Second Edition (не рекомендуется)
  • Windows® ME (не рекомендуется)
  • Windows® NT 4.0 +Service Packs 6 (не рекомендуется)
  • Windows® 2000 Professional +Service Packs 4
  • Windows® XP Professional or Home Edition +Service Packs 2 (рекомендуется)
  • Windows® Server 2003 Enterprise Edition +Service Packs 1 (рекомендуется)

Для работы системы "АвтоПланирование" требуется установка следующего программного обеспечения:
  • Firebird Server Win32 build, v.1.5.3.4870 (входит в поставку)
  • Microsoft Internet Explorer 4 и выше (необходим для справочной системы)

Аппаратные средства
  • Процессор семейств Intel® (Pentium® / Celeron® / XeonT), AMD (AthlonT / DuronT) тактовая частота которых составляет не менее 200 МГц.
    Работа на процессорах ADM K5 и AMD K6 не поддерживается.

  • Минимальные требования к оперативной памяти зависят от используемой операционной системы: для Windows XP/2000/NT 4.0 - 64 Мб; Windows Me/98 - 32 Мб. При работе в многопроцессорных системах требуется дополнительно 16 Мб оперативной памяти для каждого дополнительного процессора.

  • Видеоплата и монитор с разрешением не менее 1024х768 точек, глубина цвета не менее 16bit.
    Рекомендуемые параметры: разрешение 1280х1024 и более, глубина 16bit и более, мелкий шрифт, стандартная или классическая цветовая схема.

  • Свободное место на жестком диске не менее 30 Mb.

  • Свободный USB-порт на сервере.

  • Клавиатура.

  • Манипулятор мышь.

  • Для сетевого варианта наличие локальной вычислительной сети. Установленный протокол TCP/IP.

  • CD-ROM привод (обязателен для установки и активации системы на сервере).
    Внимание! Активация может не пройти в автоматическом режиме на некоторых типах приводов CD-ROM/R/RW и DVD-ROM/R/RW. Свяжитесь с технической поддрежкой для консультации по данному вопросу.

Версия 2.0  [октябрь 2002]

  • Добавлена возможность более гибкой настройки шаблонов. Теперь помимо цветовой схемы можно еще и изменять текст внутри шаблона.

  • Добавлена возможность просмотра связанных задач. Теперь, например, находясь в сетке День Вы можете также видеть задачи из сеток Неделя и Месяц.
